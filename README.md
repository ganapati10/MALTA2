* To run the Threshold vs Noise plot use `threshold_vs_noise.cpp` script and run it using command: `root -l threshold_vs_noise.cpp`
* To make the plot for threshold pixel map of differnece between before and after irradiation use`threshold_pixel_map_study.ipynb`script. Run this code in the SWAN.
* To make the plot for d(Threshold)/dITHR, d(Threshold)/dIDB and d(Threshold)/dICASN use `derivative_scan.C` script. To run this use command: `root -l derivative_scan.C`
* To make the plot for *Noise pixel map* of differnece between before and after irradiation use `threshold_pixel_map_study_forNoise.ipynb` script. Run this code in the SWAN.
* To make the plot for d(Noise)/dITHR, d(Noise)/dIDB and d(Noise)/dICASN use `derivative_scan_noise.C` script. To run this use command: `root -l derivative_scan_noise.C`
